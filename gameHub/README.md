# GameHub

GameHub 是一个展示游戏信息和预告片的前端应用程序。

## 已实现功能

- 卡片式布局展示游戏库
- 按照平台/主题/排序方式/输入名称精准查询
- 查看游戏详情
- 查看游戏预告片
- 查看游戏评分

## 待实现功能
- 用户管理
- 个人游戏库管理
